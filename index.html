Getting Started

Simple Installation Process

Download Module

Download the zip file from the Odoo App Store or your customer portal.

Extract to Addons

Extract the module folder into your Odoo server's custom addons directory.

Update App List

Enable debug mode and click "Update App List" in the Apps menu.

One-click Install

Search for "OmniSync Pro" and click the Install button. Done!
